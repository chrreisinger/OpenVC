\chapter*{Kurzfassung}
\chaptermark{Kurzfassung}
VHDL ist eine stark typisiert Sprache die zur Modellierung von digitalen Schaltungen wie CPUs, GPUs und
anderer verwendet wird, und ist neben Verilog die am stärksten verbreitetste.

Wie für viele andere Tools in der Electronic Design Automation (EDA) \todo{Abkürzungsverzeichnis} Industrie, sind 
für VHDL keine modernen Open Source Compiler und andere Tools vorhanden. Mit GHDL \todo{Abkürzungsverzeichnis} gibt es zwar einen Compiler,
der aber auf GCC aufsetzt und in ADA geschrieben ist, mit allen den damit vorhanden Nachteilen. 

Da mehr und mehr Sprachen auf der Java virtual machine (JVM) implementiert wurden, wurde diese auch aufgrund der später erläutertenden 
Vorteile als Ziel verwendet. Diese Arbeit präsentiert das Design und die Implementierung dieses Compilers der eine 
Teilmenge von VHDL in JVM Byte Code übersetzen kann.
